# The Toy POS System 🧸

ระบบบริหารจัดการจุดขาย (Point of Sale) และจัดการสต๊อกสินค้าที่ออกแบบและพัฒนาขึ้นมาเป็นพิเศษสำหรับร้าน **"The Toy"** โดยเน้นการรองรับโมเดลธุรกิจที่มี **"เจ้าของสินค้าหลายราย" (Multi-Vendors/Consignment)** และมีความซับซ้อนในการจัดการระบบการเงินรายวัน

---

## 🌟 ฟีเจอร์หลัก (Key Features)

### 1. ระบบจัดการหลายเจ้าของ (Multi-Owner Management)
- รองรับการฝากขายจากเจ้าของสินค้าหลายราย
- กำหนดอัตราส่วนแบ่งกำไร (GP Rate - Gross Profit) ให้กับเจ้าของแต่ละคนได้อิสระ
- สรุปยอดขาย, รายได้สุทธิ, และยอดเงินคงเหลือที่เบิกได้แบบแยกรายบุคคลแบบ Real-time
- **ระบบเบิกเงิน (Withdrawals):** บันทึกประวัติการเบิกเงินของเจ้าของสินค้าแต่ละราย และนำไปหักลบกับยอดรายได้สุทธิอัตโนมัติ

### 2. ระบบจัดการสต๊อกแบบแยกโซน (Dual-Zone Stock Management)
- **ตู้เก็บของ (Storage Qty):** สต๊อกหลักสำหรับรับสินค้าเข้า
- **ชั้นหน้าร้าน (Front Qty):** สต๊อกสำหรับวางขายจริง
- ฟังก์ชันการย้ายสต๊อกไปมาอย่างยืดหยุ่นด้วยระบบแสกนบาร์โค้ด:
  - รับของเข้าตู้ (Receive)
  - เติมของหน้าร้าน (Transfer to Front)
  - ดึงของกลับเข้าตู้ (Return to Storage)
  - ปรับลดยอด (Adjust/Reduce)

### 3. ระบบปิดยอดบัญชีประจำวัน (Daily Reconciliation)
เป็นหัวใจหลักของระบบ เพื่อให้การเงินโปร่งใสและตรวจสอบได้ 100%:
- **คำนวณยอดเงินที่ควรมี (Expected Revenue):** อ้างอิงจากยอดขาย - ส่วนลด
- **ระบุยอดเงินจริง (Actual Cash & Transfer):** พนักงานกรอกยอดเงินสดและเงินโอนที่นับได้จริง
- **จัดการส่วนต่าง (Difference Tracking):** 
  - หากเงินขาด/เงินเกิน ระบบจะบังคับให้ระบุ "สาเหตุ/หมายเหตุ" (Difference Note) เช่น ลูกค้าให้ทิป, ทอนเงินผิด
- **บันทึกของเสียและส่วนลด (Defects & Discounts):** แจกแจงให้เห็นชัดเจนว่าในวันนั้นมีมูลค่าของเสียและส่วนลดเท่าไหร่
- **ยอดยกไป (Carry Forward Cash):** ระบบนำยอดเงินทอนคงเหลือในเก๊ะ ยกยอดไปเป็นเงินทอนเริ่มต้นในวันถัดไปอัตโนมัติ

### 4. หน้าจอภาพรวมผู้บริหาร (Executive Dashboard)
- แสดงสรุปยอดขาย, ต้นทุน, กำไร, และค่าใช้จ่ายทั้งหมดในเดือนปัจจุบัน
- กราฟแสดงแนวโน้มการขายรายวัน
- การ์ดสรุปรายการส่วนต่างเงินสด ที่สามารถคลิกดูรายละเอียดเชิงลึก (รวมถึงของเสีย, ส่วนลด, และหมายเหตุ) ได้ทันที
- แจ้งเตือนสินค้าใกล้หมดสต๊อก

### 5. ระบบความปลอดภัยและบันทึกประวัติ (Security & Audit Trail)
- **ระบบ Audit Trail ขั้นสูง:** บันทึกทุกการเคลื่อนไหวสำคัญในระบบ เช่น การลบ/แก้ไขสินค้า, การจัดการเงิน, การสำรองฐานข้อมูล
- **บันทึกละเอียด (Data Comparison):** แสดงข้อมูล "ก่อนเปลี่ยน" และ "หลังเปลี่ยน" แบบ JSON เพื่อการตรวจสอบย้อนหลังที่แม่นยำ
- **ความปลอดภัย (Security Hardening):** 
    - บันทึกประวัติด้วยชื่อสิทธิ์ (Role Name) แทน ID เพื่อปกปิดโครงสร้างฐานข้อมูล
    - ติดตามการเข้าใช้งาน (Login/Logout Audit) ทุกครั้ง
    - ปกปิดข้อมูลละเอียดอ่อน (เช่น Password Hash) ในบันทึก Log

### 6. UI/UX สวยงาม ทันสมัย 
- รูปแบบปุ่มแบบโค้งมน (`btn-pill`) และมีเงา (`shadow-sm`) ทันระบบ
- ตารางข้อมูล (DataTables) ดีไซน์พรีเมียม (Premium Theme) อ่านง่าย ชัดเจน 
- ระบบแจ้งเตือนและยืนยันการทำงานผ่าน SweetAlert2 ที่ปรับแต่งปุ่มให้เข้ากับธีม

## 📋 แผนการดำเนินงานและสิ่งที่ทำสำเร็จแล้ว (Roadmap & Checklists)

- [x] **โครงสร้างฐานข้อมูลพื้นฐาน:** สร้างตารางผู้ใช้, สินค้า, เจ้าของสินค้า
- [x] **ระบบขายสินค้าหน้าร้าน:** ตัดสต๊อก, คำนวณส่วนลด, สรุปยอดเงิน
- [x] **ระบบจัดการเงินสดยกมา (Carry Forward):** ให้เงินทอนวันก่อนหน้า โอนมาเป็นเงินเริ่มต้นในวันถัดไป
- [x] **ระบบเบิกเงินเจ้าของสินค้า:** บันทึกประวัติและผูกกับกำไร (GP) ที่ควรได้
- [x] **ระบบตรวจสอบสาเหตุส่วนต่าง (Reconciliation Audit Log):** เพิ่มช่องกรอกหมายเหตุส่วนต่าง, บันทึกยอดของเสีย/ส่วนลด เพื่อการตรวจสอบย้อนหลังได้อย่างโปร่งใส
- [x] **ปรับปรุงดีไซน์ (UI Refactoring):** ใช้ `btn-pill` และปรับตารางให้สวยงามทันสมัยทั้งระบบ
- [x] **ระบบ Audit Trail แบบละเอียด:** บันทึกทุกการกระทำของผู้ใช้ พร้อมระบบความปลอดภัยในการเก็บ Log (Role Obfuscation)
- [ ] **ระบบ Export รายงาน:** เพิ่มความสามารถในการ Export รายงานต่างๆ เป็น PDF หรือ Excel
- [x] **ปรับปรุงประสิทธิภาพบนมือถือ (Mobile Responsive UI):** ปรับปรุงโครงสร้าง Form input-group, scrollable tab navigation, stacked filter และตารางให้พนักงานขายใช้งานบน Mobile & Tablet ได้ลื่นไหลยิ่งขึ้น
