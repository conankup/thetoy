# Yencha Management System 🍵

ระบบบริหารจัดการร้าน Yencha (เวอร์ชันปรับปรุงใหม่) ที่เน้นความสวยงามระดับพรีเมียม (The Toy Style) และระบบตรวจสอบสต็อกที่แม่นยำแต่ใช้งานง่าย

## 🌟 ฟีเจอร์หลัก (Key Features)

### 1. ระบบจัดการกะและเข้างาน (Shift & Attendance)
- **Photo Stamp:** ถ่ายรูปพนักงานตอนเข้า-ออกงานเพื่อความโปร่งใส
- **Late/Early Detection:** คำนวณสถานะการมาสายหรือกลับก่อนอัตโนมัติตามช่วงเวลากะที่กำหนด
- **Sealing Machine Tracking:** บันทึกเลขเครื่องซีนแก้วเพื่อใช้ตรวจสอบยอดขายจริง

### 2. ระบบสต็อก 2 โซน (Two-Zone Stock Control)
- **Storage (หลังร้าน):** เก็บเป็นหน่วยใหญ่ เช่น ถุง, ขวด, แถว
- **Front (หน้าร้าน):** เก็บเป็นหน่วยย่อย เช่น กรัม, มล., ใบ
- **Visual Estimation:** พนักงานสามารถกะระดับวัตถุดิบหน้าร้าน (เช่น เหลือครึ่งขวด = 0.5) เพื่อให้ระบบคำนวณต้นทุนได้รวดเร็ว

### 3. ระบบความปลอดภัยและการตรวจสอบ (Security & Audit)
- **Yencha Audit Logs:** บันทึกทุกความเคลื่อนไหวในระบบ (ใครทำอะไร เมนูไหน เวลาใด)
- **Reconciliation Report:** ระบบเปรียบเทียบยอดขายจากเครื่องซีนแก้ว vs สต็อกที่หายไป vs ยอดเงินที่ได้รับจริง

---

## 🏗 โครงสร้างฐานข้อมูล (Database Schema)

- `yencha_ingredients`: ข้อมูลวัตถุดิบและปริมาณต่อหน่วย
- `yencha_shifts`: ข้อมูลกะการทำงาน (เลขเครื่องซีน, ยอดเงิน)
- `yencha_attendance`: บันทึกเวลาเข้างานพร้อมรูปถ่าย
- `yencha_audit_logs`: ประวัติการใช้งานระบบ

---

## 🛠 ข้อมูลทางเทคนิค (Technical Info)

- **UI Framework:** Vanilla CSS (The Toy Theme), Bootstrap 4
- **Frontend Logic:** jQuery, SweetAlert2, Toastr, WebCam.js
- **Backend:** PHP 8+ (PDO), MySQL
